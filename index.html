Красный, белый и чёрный стиль подчёркивает характер сервера: агрессивный визуал, чистая подача и сильный акцент на контенте.

Для кого

Для авторов на YouTube и стримеров Twitch, которым интересны RP, SMP и совместный контент.

Формат

Игровой мир с ролеплеем, сюжетами, взаимодействием между участниками и ивентами.

Визуал

Лаконичный тёмный дизайн с красными акцентами и фирменным логотипом.

Гайд по вставке в GitHub Pages

  1. Создай репозиторий на GitHub, например hit-project-site.
  2. Добавь в корень репозитория файлы index.html и logo.jpg.
  3. Скопируй HTML-код сайта в файл index.html.
  4. Если логотип называется иначе, замени строку src="logo.jpg" на своё имя файла.
  5. Открой Settings → Pages, выбери ветку main и папку / (root).
  6. Сохрани настройки и подожди 1–5 минут — сайт появится по адресу GitHub Pages.
